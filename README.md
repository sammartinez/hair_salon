# _Hair Salon!_

##### _{Let's the user add stylists to their app along with adding clients to each stylists}, {August 21st, 2015}_

#### By Sam Martinez

## Description

This application is designed to allow a user (or hair salon owner) add in their stylists to their webpage and also add a client to each individual stylists without them crossing over to other stylists.

## Setup

* Run composer install
* Run MySQL on the client machine
* Run php -S localhost:8000 in order to have the machine launch the website
* Use MAMP in order to have the apache server running


## Technologies Used

* PHP
* PHPUnit
* MySQL
* HTML
* CSS
* Silex
* Twig
